# Security Policy (Email for security report needed)

## Our Commitment
At **Turbotic AI**, we are committed to maintaining a secure orchestration platform. We actively monitor and patch vulnerabilities in both our Community and Commercial editions.

## Supported Versions
We provide active security support for the following versions:

| Version | Status | Security Support |
| :--- | :--- | :--- |
| **Community Edition (CE)** | ✅ Active | Patches merged via GitHub. |
| **Enterprise** | ✅ Active | Managed updates. |
---

## Reporting a Vulnerability
**Please do not open a public GitHub Issue for security-related bugs.** To report a vulnerability:
1. **Email us:** Send a detailed report to **support@turbotic.com**.
2. **Review:** Our core team will review your report within 24 hours.
3. **Patching:** If validated, we will develop a patch. We may ask you to review the fix before it goes public.
4. **Bounty:** While we do not have a formal monetary bounty for CE yet, we provide public attribution and "Top Contributor" status to researchers who help us secure the platform.
